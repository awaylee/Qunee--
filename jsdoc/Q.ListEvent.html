Q.ListEvent

new ListEvent(source, kind, data, index, oldIndex)

集合变化事件，添加，删除元素，调整元素次序或者清空集合时，派发该类事件
Parameters:
Name Type Description
source Object 集合对象
kind Object 事件种类: Q.ListEvent.KIND_ADD, Q.ListEvent.KIND_REMOVE, Q.ListEvent.KIND_CLEAR, Q.ListEvent.KIND_INDEX_CHANGE
data Object 事件数据，比如元素添加事件时，这个参数为新添加的元素或者元素集合
index int 序号序号，比如元素添加事件时，插入的位置
oldIndex Object 之前的序号，更改元素次序事件时，此参数表示元素原来的位置
