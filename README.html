Qunee for HTML5 README

Qunee for HTML5 is an extensive HTML5/JavaScript class library that provides diagram visualization controls, data structures, and functionality to automatically calculate layouts for graphs, diagrams, and networks.

How to Start

If you are using this software for the first time, you might be interested in the following information:

  • What are the requirements for developing and deploying applications?

    Development

    We suggest to use a modern IDE, e.g., IntelliJ WebStorm, IntelliJ IDEA, Netbeans, or Eclipse.

    Deployment

    To run Qunee for HTML5 applications, including the provided demo applications, you will need to have an HTML5-capable browser.
    At the time of its release, this Qunee for HTML5 release has been tested with the following browsers:

    • Internet Explorer 9+
    • Firefox
    • Chrome
    • Safari
    • Opera

  • What demos, and documentation are included?

    A number of resources are included to help you learn more about using Qunee for HTML:

    Included Resources

    • The documents/ directory contains the more advanced, comprehensive Developer's Guides which describe the architecture and the main concepts used in Qunee
    • The demos/ directory contains extensive source code demo applications.
    • The demos/lib/qunee-min.js file is the Qunee HTML5 core library file
