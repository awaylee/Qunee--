Class: ListEvent

Q.ListEvent

new ListEvent(source, kind, data, index)

Collection change events, distribute this kind of events in case if to add or delete elements, adjust element order or clear collection
Parameters:
Name Type Description
source Object Collection object
kind Object Event kind: Q.ListEvent.KIND_ADD, Q.ListEvent.KIND_REMOVE, Q.ListEvent.KIND_CLEAR, Q.ListEvent.KIND_INDEX_CHANGE
data Object event data. In case of element adding event, this parameter will be newly added element or element collection
index int serial number. In case of element adding event, insert the serial number in front of position
oldIndex. Object If to modify the element sequence event, this parameter indicates the original position of element
